Мастерская Сары находится в отреставрированном шахтёрском домике, где она по совместительству живёт. Здесь есть все необходимое, что только нужно мастеру-керамисту: два широких стола, различные системы для хранения и сушки изделий, печь для обжига, гончарный круг.
